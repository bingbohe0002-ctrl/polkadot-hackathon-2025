# Life++ Frontend Dashboard

Frontend management console for Life++ PoC system visualization and management.

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

Frontend will start at http://localhost:5173

## Build

```bash
npm run build
```

## Environment Variables

Create `.env` file:

```
VITE_API_URL=http://localhost:3000
```

## Feature Modules

- **Dashboard**: System overview and real-time monitoring
- **Regulatory**: Regulatory review workflow
- **Proofs**: Proof browsing and search
- **Agents**: Agent registration and management
- **Validators**: Validator network monitoring
- **ChainRank**: Reputation score analysis
- **Compliance**: KYC/AML compliance monitoring
- **Economics**: Token economics
- **Robot Control**: Real-time robot control
- **Settings**: System configuration

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Lucide React Icons
